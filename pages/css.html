An Introduction to CSS

HTML provides a webpage's basic content, but CSS controls how that content is displayed. The rules for CSS can be complicated and arbitrary. For this reason, it is ultimately more important to understand how to learn about CSS than to memorize particular rules. This set of exercises will help you learn how to learn about CSS.

Learning About CSS

1. Shorthands

Over time CSS has developed a number of shortcuts to allow developers to save time while designing webpages. For instance, take a look at the box below:

The CSS for this box could be written like this:

.box-simple {
	border-width: 1px;
	border-style: dashed;
	border-color: black;
	width: 150px;
	height: 75px;
}

However, to simplify things, we can use a shorthand to write the CSS like this instead:

.box-simple {
	border: 1px dashed black;
	width: 150px;
	height: 75px;
}

Notice that the three border properties, border-width, border-style, and border-color are combined together into one border property. CSS is full of so many shortcuts like this that it is futile to try to memorize everything (especially as a beginner!). The crucial skill to learn is how to research CSS syntax online. (Trust us, this is a life-long skill employed daily by all front-end developers!!!)

Task 1.1: Research the CSS border property on the W3Schools and Mozilla Developer Network website. Which one do you find more helpful? Why?

Task 1.2: Try doing a Google search yourself for the CSS border property. Can you find any useful resources?
