2. jQuery

Last week you might have also noticed that the syntax of JavaScript is a bit complicated. jQuery is a powerful toolkit that makes working with JavaScript easier. To include jQuery with our applications we first need to include the following script tag in our page's head:



<script src="http://code.jquery.com/jquery-1.11.2.min.js"></script>

Make sure that you include this tag before the rest of your JavaScript.

Simple Examples

jQuery gives us access to simplified JavaScript syntax, based around the '$' character, as well as the CSS selectors which we are used to. For instance:


$('p')

would select all of the paragraphs in a document and:


$('p').html('Hello World');

would change the text in every paragraph on the page to the words "Hello World". If we wanted to atach this event to a button we could do something like:


$('#my-button').on('click', function() {

  $('p').html('Hello World');

})

That function() { ... } block is called an anonymous function - think of it as simply a wrapper to enclose all fo the code which you wish to run whenever the button is clicked. The '#my-button is exactly like a CSS selector - it identifies the relevant button by its 'id' attribute in the HTML.

Finally, we need to wrap the whole thing in a 'ready' event listener to make sure none of the code is run before our HTML document is full loaded. The end result will look like this:


$('document').ready(function() {
  
  $('#my-button').on('click', function() {

    $('p').html('Hello World');

  })

})

At this point it's probably easier to look at an example of everything. Do take a minute and check out the source code.
