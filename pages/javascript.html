2. JavaScript

Last week we looked at frameworks and some fairly advanced HTML and CSS. As you move into your final websites, this week gives us a chance to look at one more piece of the puzzle - JavaScript - which you can use for creating your final website. The JavaScript we're going to look at this week is incredibly basic, but hopefully can set you up for some learning on your own.

A Hello World

The simplest program in any programming language is one that says "Hello World". We can do a "hello world" program in JavaScript fairly easily. The first thing we need to do is create a basic HTML page.



<html>

  <head>
    <title>JavaScript Hello World</title>
  </head>

  <body>

    <h1>A Quick Introduction to JavaScript</h1>
    <p>This page provides a very simple overview of a hello world program in JavaScript.</p>
    <button>Click Me!</button>

  </body>

</html>

Visit the page here. What happens when you click the button? Nothing right? Well, let's fix that. First we need to add a <script> tag.



<html>

  <head>
  
    <title>JavaScript Hello World</title>
  
    <script>
      
    </script>
  
  </head>

  <body>

    <h1>A Quick Introduction to JavaScript</h1>
    <p>This page provides a very simple overview of a hello world program in JavaScript.</p>
    <button>Click Me!</button>

  </body>

</html>

Now inside that script tag we're going to include what's known as a function. A function is simply a little block of code that's meant to be run all at once. Inside that function we simply use JavaScript's built-in alert function to display our message.



<html>

  <head>
  
    <title>JavaScript Hello World</title>
  
    <script>
      function helloWorld() {
        alert('Hello World!');
      }
    </script>
  
  </head>

  <body>

    <h1>A Quick Introduction to JavaScript</h1>
    <p>This page provides a very simple overview of a hello world program in JavaScript.</p>
    <button>Click Me!</button>

  </body>

</html>

But wait, that's still not quite enough. The reason is that the webpage doesn't know when you run our function. To change that we have to add an onclick property to the <button> element.



<html>

  <head>
  
    <title>JavaScript Hello World</title>
    <script>
      function helloWorld() {
        alert('Hello World!');
      }
    </script>
  
  </head>

  <body>

    <h1>A Quick Introduction to JavaScript</h1>
    <p>This page provides a very simple overview of a hello world program in JavaScript.</p>
    <button onclick="helloWorld()">Click Me!</button>

  </body>

</html>

Try it out here.

Multiple Functions

In the last example we only had one function. However, it's completely possible to have multiple functions that are run by different buttons of your webpage. Here is an example:



<html>

  <head>

    <title>Multiple Functions</title>
    <script>
      
      function exampleOne() {
        alert('This is the first function!');
      }

      function exampleTwo() {
        alert('This is the second function!!!');
      }

    </script>

  </head>

  <body>
  
    <h1>Each Button Runs Its Own Function</h1>
    <button onclick="exampleOne()">First Function</button>
    <button onclick="exampleTwo()">Second Function</button>
  
  </body>

</html>

Try it out here.

Editing HTML

At this point you might be wondering if we can do anything better with JavaScript than simply making annoying popup windows. Well, luckily, the answer is yes! Let's look at how we can edit some HTML.



<html>
  
  <head>
  
    <title>Editing HTML</title>
  
    <style>

      p#target {
        border: 1px solid black;
        border-radius: 5px;
        height: 40px;
        width: 500px;
        padding: 5px;
        background-color: navy;
        color: white;
      }

    </style>

    <script>

      function editHtml() {
        document.getElementById('target').innerHTML = 'Yes. We changed it!';
      }
      
    </script>

  </head>

  <body>
    <h1>An Example of How to Change HTML</h1>
    <p id="target">This is the target of our program.</p>
    <button onclick="editHtml()">Click Here</button>
  </body>

</html>

Can you see how the program works? First it finds the element based on it's id attribute (<p id="target">) and then it simply sets its innerHTML property to the desired value. You can see it in action here.
